# Приложение для заметок и организации

Этот репозиторий содержит документацию и прототип простого приложения
для заметок и личного органайзера.

## Описание приложения

Приложение позволяет пользователю:
- создавать и редактировать заметки;
- удалять заметки;
- просматривать список всех заметок;
- открывать отдельную заметку для просмотра;
- делиться заметками;
- синхронизировать заметки с облаком (при наличии подключения к интернету);
- хранить данные заметок в зашифрованном виде.

## Содержимое репозитория

- `SRS.docx` — спецификация требований к системе  
  (функциональные требования: создание/редактирование заметок;  
  нефункциональные: шифрование данных и др.).
- `Use case.puml` / `Use case.png` — UML-диаграмма **Use Case**  
  (создание и обмен заметками).
- `Activity sync.puml` / `Activity sync.png` — UML-диаграмма **Activity**  
  (синхронизация заметок с облаком).
- `prototype_main.png`, `prototype_menu.png` — скриншоты прототипа из Figma  
  (главный экран и меню заметок).
- `Figma prototype.txt` — ссылка на интерактивный прототип в Figma.
- `git_commands.txt` — список использованных команд Git.
